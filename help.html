Overview

The 2 FTS Ballast Weight App is a planning tool/aid for Aircraft Commanders to check ballast requirements for single-passenger and multiple-passenger scenarios.

It uses the aircraft data packaged with the app to compare payload against aircraft weight limits and to show whether ballast is required, permitted, or not allowed.

The app is for information only.
Full responsibility remains with the Aircraft Commander.

Features

  • Single-passenger mode checks for payload, aircraft all-up mass, ballast, and approach speed.
  • Alternative aircraft comparison for the same single-passenger payload.
  • Multiple-passenger summary showing each passenger's ballast possibilities.
  • Optional aircraft allocation using aircraft-specific commander and ballast settings.
  • PDF export of the multiple-passenger summary.

How to use Single Mode

  1. Select the desired aircraft.
  2. Enter the Aircraft Commander weight with parachute.
  3. Enter the passenger weight with parachute.
  4. Select how many ballast weights are fitted.
  5. Review the calculations, output checks, and alternative aircraft table.

How to use Multiple Mode

  1. Set the number of passengers you will be entering.
  2. Enter each passenger name and weight with parachute.
  3. If required, enable Set Aircraft Ballasts, choose the available aircraft, then enter each Aircraft Commander weight and ballast setting.
  4. Review the summary table for required ballast, permitted ballast, notes, and any aircraft allocation.
  5. Use Download PDF if you need an exportable summary.

Assumptions

  • In single-passenger mode, the Aircraft Commander is assumed to be in the rear seat and the passenger in the front seat.
  • Single-passenger mode assumes nil carry-on items.
  • Passenger and commander weights are expected to include parachutes.
  • Outputs depend on the aircraft data currently bundled with the app.

Troubleshooting / Contact

  • If the aircraft list does not load, refresh the page and try again.
  • If PDF export is unavailable, reload the page so the local PDF libraries can load again.
  • If a weight or ballast output looks unexpected, check that the entered values include parachutes and that the correct aircraft is selected.
  • If the underlying aircraft data needs updating, use the project source repository to raise an issue or propose a change.
    • For any feedback or suggestions, you can contact Brad Kennedy using this contact form.
